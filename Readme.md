# HW or Hotel Web

HW is a static website built using CSS3 and HTML.
Current implementation is responsive. It does not use media queries, flex box and grids in CSS3.
This can be considered as initial project for CSS3 tutorial.

## How to run

- Clone the repo
- Run index.html
